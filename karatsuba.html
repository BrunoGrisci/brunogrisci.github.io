How to use this visualizer

This tool compares classroom multiplication and Karatsuba multiplication step by step.

How to use:

  • Enter two integers (they are padded automatically to power-of-two lengths when needed).
  • Use Run step, Auto run, or Run full to control execution.
  • Switch tabs to compare algorithms; counters show one-digit multiplications and sums so far.
  • Use panel toggles to free space and inspect the decomposition tree.
